Network Tests

These test pages measure the performance traits of your browser. Your browser must support JavaScript in order to run the tests. The following browser performance traits are measured.

CVSNO: echo genFixBrowser($_SERVER['HTTP_USER_AGENT']);
  1. How many connections are opened per hostname (for HTTP/1.1)?
  2. How many total connections are opened across all hostnames (for HTTP/1.1)?
  3. Do external scripts block other downloads?
  4. Do stylesheets block other downloads?
  5. Do inline scripts after stylesheets block other downloads?
  6. Are resources with a future Expires header cached?
  7. Are page redirects with a future Expires header cached?
  8. Are resource redirects with a future Expires header cached?
  9. Do prefetch links work?
  10. Is compression supported?
  11. Are data: URLs supported?
